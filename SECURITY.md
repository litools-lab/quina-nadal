# Política de seguretat

| Versió | Suportada |
|--------|-----------|
| 1.0.x | ✅ |

## Reportar una vulnerabilitat

Si trobes una vulnerabilitat de seguretat, **no obris un Issue públic**.

En lloc d'això, utilitza la funcionalitat de **[Private vulnerability reporting](../../security/advisories/new)** de GitHub, descrivint:

1. El tipus de vulnerabilitat
2. Els passos per reproduir-la
3. El possible impacte

Respondrem en un termini de 48 hores i t'informarem del procés de correcció.

> Nota: Atès que és una aplicació web estàtica sense backend ni gestió de dades d'usuari, la superfície d'atac és molt limitada.
